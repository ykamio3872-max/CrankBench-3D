# CrankBench-3D v1.0.0 - 3Dエンジン動作シミュレータ -
## 1. プロジェクト概要
- 以前作成した、2Dアニメーションを使用したエンジンシミュレータ（CrankBench）をベースに、バックエンドで行っていた熱力学的計算（単室モデル・Wiebe関数を用いた疑似燃焼モデル）を引き継いで、フロントエンドでの描画処理をUnityを使用した3DCGにアップグレードすることを試みています。前回のプロジェクトでは、`TypeScript + React + Rust + Tauri`というパッケージでアプリをビルドしていましたが、今回のプロジェクトではUnityをベースにしたアプリとしてビルドを行う予定です。
